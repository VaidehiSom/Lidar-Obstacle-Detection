# Obstacle Detection using LIDAR

<img src="media/ObstacleDetectionFPS.gif" width="700" height="400" />

Versions used are as follows:

* Ubuntu 16.04
* PCL - v1.7.2
* C++ v11
* gcc v5.5

## Build


   ```shell
   sudo apt install libpcl-dev
   mkdir build && cd build
   cmake ..
   make
   ./environment
   ```
